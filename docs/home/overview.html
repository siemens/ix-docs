Welcome to Siemens Industrial Experience

Siemens Industrial Experience is an open-source design system for designers and developers to consistently create the perfect digital experience for partners and customers within industrial contexts. It gives you access to a broad range of UI patterns, web-based code implementations, design tools, resources and content guidelines to accelerate collaboration between designers, developers and writers, helping to build efficient and consistent products.

Welcome to Siemens Industrial Experience

Guidelines
Information such as design principles, UX writing and conversational design or mobile experience.
Components
Usage guidelines and a code tab to find component previews and code examples for all supported frameworks.
Icons
Icon library with usage guidelines and code snippets to integrate icons directly into your code.
Styles
Style definitions including themes, colors, borders, shadows, elevation and typography.

Figma library​

Our Figma library contains all tokens, components and blueprints ready for app design.

  • Access: We maintain the primary library in the Siemens AG Global Figma account. Siemens employees with a Figma license can search for "iX Design System".
  • Guest access and classic theme: Guest access and the classic (open-source) library are available upon request — please contact us.
  • Brand and variables: The library includes Siemens brand assets and Figma variables for easy theming. See Designing with themes for details.
